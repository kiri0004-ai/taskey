# Taskey
Taskey is a basic task management application that allows users to create, view, edit, and delete tasks. Each task has attributes such as title, description, priority, status, and timestamps. Users can organise tasks into projects for better management.

## Maestro Framework
Taskey is built using the Maestro Framework. Maestro is a lightweight PHP framework that follows the MVC architectural pattern, making it easy to develop and maintain web applications.
